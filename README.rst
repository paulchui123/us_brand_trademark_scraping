TESS brand trademark script
===========================

TESS --> Trademark Electronic Search System

search site uspto.gov <https://www.uspto.gov/trademark>
search trademark <http://tmsearch.uspto.gov/bin/gate.exe?f=tess&state=4805:q3gpxn.1.1>
